
# Neovim 高阶技巧指南
### 核心寄存器类型
| 寄存器    | 用途           | 示例             |
| ------ | ------------ | -------------- |
| `""`   | 默认无名寄存器      | `yy` → `p`     |
| `"a-z` | 命名寄存器（26个）   | `"ayy` → `"ap` |
| `"_`   | 黑洞寄存器（不保存内容） | `"_dd` 永久删除    |
| `"+`   | 系统剪贴板        | `"+y` 复制到系统    |
## 替换模式实战
### 快速修改单个字符
1. Normal Mode下将光标移至目标字符
2. 按`r`键进入替换模式
3. 输入新字符自动返回Normal Mode

**场景示例**：
```python
# 原内容：print("helxo world")
# 光标在x位置按r → l → 自动修正
print("hello world")
```

##  宏操作(用于提升效率)

### 基础操作流程
1. **开始录制**：`qa`（记录到a寄存器）
2. **执行操作**：任意编辑动作
3. **结束录制**：再按`q`
4. **调用宏**：`@a` 或 `10@a`重复10次



## 启动参数妙用
### 常用参数表
| 参数             | 作用          |
| -------------- | ----------- |
| `nvim -O`      | 左右分屏打开多个文件  |
| `nvim -d`      | 差异对比模式      |
| `nvim --clean` | 纯净模式（不加载配置） |

### 故障修复指南
**场景**：配置错误导致无法启动
```bash
# 步骤1：使用纯净模式打开配置文件
nvim --clean ~/.config/nvim/init.lua

# 步骤2：修复错误配置后正常启动
nvim
```
