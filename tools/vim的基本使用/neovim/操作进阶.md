## Text Object 核心概念
**定义**：Vim/Neovim中表示特定文本结构的操作单元，用于精准操作（复制/删除/修改）目标文本

**操作公式**：
```vim
<操作符><介词><对象>
```
- **操作符**：`d`(删除)/`y`(复制)/`c`(修改)
- **介词**：
  - `i`：inner（不含边界）
  - `a`：around（含边界）
- **对象**：
  - `w`(word)/`"`(引号)/`(`(括号)/`t`(标签)/`p`(段落)

---

##  常用操作速查表

| 命令    | 效果           | 示例场景                     |            |        |
| ----- | ------------ | ------------------------ | ---------- | ------ |
| `diw` | 删除当前单词（不含空格） | `hello                   | world` → ` | world` |
| `daw` | 删除单词及空格      | `[ text ]` → `[          | ]`         |        |
| `yi"` | 复制双引号内容      | `"vim"` → 复制`vim`        |            |        |
| `da{` | 删除大括号及内容     | `{ code }` → 完全删除        |            |        |
| `cit` | 修改HTML标签内容   | `<div>old</div>` → 可编辑内容 |            |        |

---

##  Word 与 WORD 区别
|        | word (`w`)       | WORD (`W`)       |
| ------ | ---------------- | ---------------- |
| **定义** | 字母/数字/下划线        | 任意非空白字符          |
| **示例** | `foo_bar` 3个word | `foo_bar` 1个WORD |
| **跳转** | `w/e/b`          | `W/E/B`          |

---

##  高效组合技
### 移动指令组合
| 命令   | 效果                        |
|--------|---------------------------|
| `d2w`  | 删除后续2个单词            |
| `y3e`  | 复制到第3个单词结尾        |
| `dtx`  | 删除直到字符x出现的位置    |

### 行级操作
```vim
yy    " 复制整行
dd    " 删除整行
ygg   " 复制到文件头
dG    " 删除到文件尾
```
