
# Neovim模式与光标移动

##  核心模式解析
### 1. Normal Mode（默认模式）
- **特性**：所有按键触发命令而非输入
- **进入方式**：
  - 启动Neovim自动进入
  - Insert Mode下按`Esc`
- **标志**：无特殊提示（左下角空白）

### 2. Insert Mode（输入模式）
- **特性**：常规文本输入
- **进入方式**：Normal Mode下按`i`
- **标志**：左下角显示`-- INSERT --`
### 模式切换示意图
```text
Normal Mode
  ⇧ Esc
  ⇩ i
Insert Mode
```

##  必学命令速查
| 命令       | 功能      | 简写   |
| -------- | ------- | ---- |
| `:write` | 保存文件    | `:w` |
| `:quit`  | 退出编辑器   | `:q` |
| `:wq`    | 保存并退出   | -    |
| `:q!`    | 强制退出不保存 | -    |

**命令模式技巧**：
1. 输入`:`进入命令模式
2. 支持命令历史（上下箭头翻查）
3. `Esc`可退回Normal Mode

##  高效光标移动
### 基础移动（Normal Mode）
```vim
h    ← 左移
j    ↓ 下移
k    ↑ 上移
l    → 右移
```

### 多行空格删除（Visual Block ）
Ctrl + q
### 进阶跳转
| 命令    | 功能     | 示例      |
| ----- | ------ | ------- |
| `10j` | 下移10行  | 快速浏览长文件 |
| `30l` | 右移30字符 | 跨域长单词   |
| `gg`  | 跳至文件首  | 快速回到开头  |
| `G`   | 跳至文件尾  | 添加新内容时  |
| `0`   | 行首     | 替代Home键 |
| `$`   | 行尾     | 替代End键  |
