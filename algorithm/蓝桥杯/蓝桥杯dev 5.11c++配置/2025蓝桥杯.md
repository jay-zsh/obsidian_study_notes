# 常见坑点
- 1 main（）函数里必须在结尾“return 0”；
- 2.一定要开 long long 
> step1:把main函数前面的int 换成signed
> step2:在代码开头写上 # define int long long;

- 3.关于输入输出的速度问题
> main函数里写上三行关流代码(暴力枚举可以得到很大提升），并在开头写上#define endl '\n'
```c++
ios::sync_with_stdio(0);
cin.tie(0)
cout.tie(0)
```
- 万能头文件 
```c++
#include <bits/stdc++.h>
```
- c++标准要选择最高的

```c++
#include <bits/stdc++.h>//万能头文件
#define int long long//将int类型转为long long
#define INF 0x3f3f3f3f3f3f3f//定义最大默认数
#define endl '\n'//将endl转为'\n'避免使用endl降低速度
using namespace std;

signed main() {
	//关流三板斧(提高输出流速度)
    ios::sync_with_stdio(0);
    cin.tie(0);
    cout.tie(0);
    
    int n,m,a;
    cin >> n;
    cin >> m;
    cin >> a;
    cout << n*m*a <<endl;

    return 0;
}
```
