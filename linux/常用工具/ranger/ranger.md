### **Ranger 核心操作速查表**

#### **1. 基础导航**
| 快捷键           | 功能说明                |
| ------------- | ------------------- |
| `j`/`k`       | 向下/向上移动光标           |
| `h`           | 返回上级目录              |
| `l` 或 `Enter` | **进入子目录/打开文件**（重要！） |
| `o`           | **打开文件排序菜单**（见下文详解） |
| `i`           | **预览文件内容**（内置查看器）   |
| `gg`/`G`      | 跳转到列表首/尾部           |

#### **2. 文件操作**
| 快捷键 | 功能说明                     |
|--------|-----------------------------|
| `Space`| 单选文件                    |
| `v`    | 进入多选模式                |
| `yy`   | 复制选中文件                |
| `dd`   | 剪切选中文件                |
| `pp`   | 粘贴文件                    |
| `dD`   | 删除文件（需确认）          |

#### **3. 高级功能**
| 快捷键   | 功能说明                          |
|----------|----------------------------------|
| `o` 展开 | **排序模式**：<br>`oa`-按名称<br>`ob`-按修改时间<br>`oc`-按大小<br>`os`-按文件类型 |
| `i` 进阶 | **预览控制**：<br>`i`-切换预览<br>`zh`-显示隐藏文件<br>`zp`-图片预览 |
| `z`      | **设置菜单**：<br>`zh`-显示隐藏文件<br>`zp`-图片预览 |

#### **4. 特殊操作**
| 组合键              | 功能说明                      |
| ---------------- | ------------------------- |
| `:` + `shell 命令` | 直接执行 Linux 命令（如 `:ls -l`） |
| `S`              | 在当前目录打开 shell             |
| `r`              | 重命名文件                     |

---

### **高频场景示例**
1. **快速打开文件**  
   - 光标定位文件 → 按 `Enter` 直接打开  
   - 按 `i` 快速预览内容（避免误打开）

2. **批量操作**  
   ```text
   3. 按 `v` 进入多选模式
   4. 用 `j`/`k` 选择多个文件
   5. 按 `dd` 剪切或 `yy` 复制
   6. 进入目标目录按 `pp` 粘贴
   ```

7. **自定义排序**  
   - 按 `o` 调出排序菜单 → 选择 `ob` 按修改时间排序

---

### **操作逻辑图解**
```text
[ 文件列表 ]
  ↑(k) ↓(j)
  Enter(l) → 打开/进入
  o → 排序 → oa/ob/oc...
  i → 预览 → 空格翻页/q退出
```

> 💡 提示：所有操作均可通过 `?` 查看帮助菜单，支持 Vim 风格的键位组合。
