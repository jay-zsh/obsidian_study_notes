### 🧭 ​**一、基础导航**​

1. ​**移动光标**​
    
    - `j`/`k`：向下/向上移动（同 Vim）
    - `h`：返回上级目录
    - `l`：进入子目录或打开文件
        
    - `gg`/`G`：跳转至列表顶部/底部
    - `Ctrl+F`/`Ctrl+B`：向下/向上翻页
2. ​**历史记录**​
    
    - `H`：后退到上一位置
    - `L`：前进到下一位置

### 🖼️ ​**四、预览与配置**​

1. ​**文件预览**​
    - 默认支持文本/代码预览
    - 图片预览需安装 `ueberzug` 并配置：
        
        bash
        
        复制
        
        ```bash
        sudo pip3 install ueberzug
        ```
        
        编辑 `~/.config/ranger/rc.conf` 添加：
        
        ini
        
        复制
        
        ````ini
        set preview_images true
        set preview_images_method ueberzug
        ```[5](@ref)  
        ````
        
2. ​**显示隐藏文件**​
    - 临时切换：按 `zh` 或 `Backspace` 键
    - 永久生效：在 `rc.conf` 中添加 `set show_hidden true`
