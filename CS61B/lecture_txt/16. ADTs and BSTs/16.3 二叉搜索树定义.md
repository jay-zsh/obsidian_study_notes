# 16.3 二叉搜索树定义
---

树由节点和连接这些节点的边构成。

**约束条件**：任意两个节点之间仅存在唯一路径。

在某些树结构中，我们会选择一个**根节点**——没有父节点的特殊节点。

树还包含**叶节点**——没有子节点的末端节点。

基于上述基本约束，我们可以引入新的约束条件来定义更具体的树类型，例如二叉树和二叉搜索树。

- **二叉树**：在满足基础树形结构的前提下，额外遵循二元属性约束——每个节点最多拥有两个子节点
    
- **二叉搜索树**：在满足所有树形约束的基础上，还满足以下核心性质：对于树中的每个节点X
    
    - 左子树所有节点的键值均小于X的键值
        
    - 右子树所有节点的键值均大于X的键值。**请牢记这一性质!! 本模块及61B课程中将频繁引用此特性**
        

![[PANG/ee6027dfc10cc82e2336f3fcb208a81f_MD5.jpg]]

有效BST示例：左子树所有键值小于父节点，右子树所有键值大于父节点

```
private class BST<Key> {
    private Key key;
    private BST left;
    private BST right;

    public BST(Key key, BST left, BST Right) {
        this.key = key;
        this.left = left;
        this.right = right;
    }

    public BST(Key key) {
        this.key = key;
    }
}
