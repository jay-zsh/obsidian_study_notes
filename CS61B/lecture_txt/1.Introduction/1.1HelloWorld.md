# 1.1 你的第一个Java程序

在介绍任何新语言时，经典的入门程序都是**Hello World**，即向控制台输出`Hello World`的程序。在Java中，**Hello World**的实现如下：

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello world!");
    }
}
```

---

相较于Python等其他语言，这段代码可能显得冗长。然而Java的这种特性有其深层原因，我们将在后续章节中探讨。现在请先注意上述代码的几个关键语法特征：

- **类声明** `public class HelloWorld`：在Java中，所有代码都存在于类中
    
- `**main**`函数：所有可执行代码必须位于`public static void main(String[] args)`方法内。关于此声明的具体含义将在后续章节详解
    
- **花括号** `{}`用于包裹代码段（函数、类以及其他将在后续章节介绍的类型）
    
- 所有语句必须以**分号**结尾
