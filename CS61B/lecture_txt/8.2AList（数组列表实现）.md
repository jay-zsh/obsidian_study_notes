### 构建AList类和构造函数
```java
public class AList<T> {  
    private T[] items;  
    private int size;
 
	
	public AList() {  
    items = (T[]) new Object[100];  
    size = 0;  
}
    
}
```

>泛型申明数组  items = (T[])new object[100]) //强制类型(T[])

### 一个辅助方法+8个功能方法

>重要知识：System.arraycopy(原始数组,复制起始位置，目标数组,粘贴起始位置,复制长度)
#### resize（功能方法）调整大小
```java
private void resize(int capacity){
    T[] a = (T[]) new Object[capacity]
    System.arraycopy(items,0,a,0,size);
    items = a;
}
```


#### addFirst

```java
/** Inserts x into the front of the list. */  
public void addFirst(T x) {  
    if (size == items.length) {  
        resize(size * 2);  
    }  
    System.arraycopy(items, 0, items, 1, size);  
    items[0] = x;  
    size++;  
}
```
#### removeFirst
```java
/** Removes item from the front of the list. */  
public void removeFirst() {  
    if (size == 0) return;  
    System.arraycopy(items, 1, items, 0, size - 1);  
    items[size - 1] = null; // Avoid memory leak  
    size--;  
}
```

#### addLast
```java
/** Inserts x into the back of the list. */  
public void addLast(T x) {  
    if (size == items.length) {  
        resize(size * 2);  
    }  
    items[size++] = x;  
}
```

#### removeLast
```java
/** Deletes item from back of the list and returns deleted item. */  
public T removeLast() {  
    if (size == 0) return null;  
    T x = items[--size];  
    items[size] = null; // Avoid memory leak  
    return x;  
}
```

#### getLast
```java
/** Returns the item from the back of the list. */  
public T getLast() {  
    if (size == 0) return null;  
    return items[size - 1];  
}
```

#### insert
```java
/** Inserts x at index dest. */  
public void insert(T x, int dest) {  
    if (dest < 0 || dest > size) {  
        throw new IndexOutOfBoundsException("Invalid index: " + dest);  
    }  
    if (size == items.length) {  
        resize(size * 2);  
    }  
    System.arraycopy(items, dest, items, dest + 1, size - dest);  
    items[dest] = x;  
    size++;  
}
```

#### get

```java
/** Gets the ith item in the list (0 is the front). */  
public T get(int i) {  
    if (i < 0 || i >= size) {  
        throw new IndexOutOfBoundsException("Index: " + i);  
    }  
    return items[i];  
}
```

#### size

/** Caching .Returns the number of items in the list. */  
public int size() {  
    return size;  
}
