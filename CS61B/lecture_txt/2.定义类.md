## 一、静态方法与实例方法

### 1.1 静态方法示例

```java
public class Dog {
    public static void makeNoise() {  // 静态方法
        System.out.println("Bark!");
    }
}
```

调用方式：

```java
Dog.makeNoise();  // 通过类名调用
```

### 1.2 实例方法示例

```java
public class Dog {
    public void bark() {  // 实例方法
        System.out.println("Woof!");
    }
}
```

调用方式：

```java
Dog d = new Dog();
d.bark();  // 通过对象调用
```

### 对比表格

|特性|静态方法|实例方法|
|---|---|---|
|声明关键字|`static`|无|
|调用方式|`类名.方法名()`|`对象.方法名()`|
|内存分配|类加载时分配|对象创建时分配|
|访问权限|只能访问静态成员|可访问所有成员|

### 2.1 基本实例化流程

```java
Dog d;          // 1.声明对象变量
```

```java

    public int weight;

    public Dog(int w) {

    }
}
```

使用方式：

```java
Dog myDog = new Dog(20);  // 创建20kg的狗
```

## 三、对象数组

### 3.1 创建对象数组

```java
Dog[] kennel = new Dog[3];     // 创建数组
kennel[0] = new Dog(8);       // 初始化元素
kennel[1] = new Dog(20);
kennel[2] = new Dog(35);
```

### 3.2 数组操作示例

```java
// 计算平均体重
double total = 0;
for (Dog d : kennel) {
    total += d.weight;
}
System.out.println("平均体重: " + total/kennel.length);
```

## 四、方法对比

### 4.1 静态方法 vs 实例方法

```java
// 静态比较方法
public static Dog compareStatic(Dog a, Dog b) {
    return (a.weight > b.weight) ? a : b;
}

// 实例比较方法
public Dog compareInstance(Dog other) {
    return (this.weight > other.weight) ? this : other;
}
```

### 对比表格

|场景|静态方法调用|实例方法调用|
|---|---|---|
|比较两只狗|`Dog.compareStatic(d1, d2)`|`d1.compareInstance(d2)`|
|适用场景|工具类方法|对象行为相关|

## 五、静态变量

### 5.1 定义与访问

```java
public class Dog {
    public static String species = "Canis lupus";  // 静态变量
    public int weight;                             // 实例变量
}
```

正确访问方式：

```java
System.out.println(Dog.species);  // 通过类名访问
```

### 5.2 对比表格

|特性|静态变量|实例变量|
|---|---|---|
|生命周期|类加载到卸载|对象创建到销毁|
|存储位置|方法区|堆内存|
|访问方式|`类名.变量名`|`对象.变量名`|

## 六、Main方法与命令行参数

### 6.1 标准main方法

```java

    public static void main(String[] args) {
        System.out.println("接收参数数量: " + args.length);
    }

```

### 6.2 参数求和实现

```java
public class SumCalculator {
    public static void main(String[] args) {
        int sum = 0;
        for (String arg : args) {
            sum += Integer.parseInt(arg);
        }
        System.out.println("参数总和: " + sum);
    }
}
```

## 七、综合练习

### 7.1 创建狗数组并筛选

```java
public class DogManager {
    public static void main(String[] args) {
        Dog[] dogs = {
            new Dog(8), 
            new Dog(15), 
            new Dog(30)
        };
        
        // 找出体重超过10kg的狗
        for (Dog d : dogs) {
            if (d.weight > 10) {
                d.makeNoise();
            }
        }
    }
}
```

```
bark. bark.
woof!
```
