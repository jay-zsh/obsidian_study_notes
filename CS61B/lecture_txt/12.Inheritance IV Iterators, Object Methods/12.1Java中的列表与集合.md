## 实际开发中的Java列表

虽然我们实现了自定义列表，但Java本身提供了`List`接口及多种实现（如`ArrayList`）。注意`List`是接口，不能直接实例化，必须选择具体实现类。

完整类名调用方式：

```java
java.util.List<Integer> L = new java.util.ArrayList<>();
```

更简洁的导入方式：

```java
import java.util.List;
import java.util.ArrayList;

public class Example {
    public static void main(String[] args) {
        List<Integer> L = new ArrayList<>();
        L.add(5);
        L.add(10);
        System.out.println(L);
    }
}
```

## 集合类型

集合是元素唯一的容器，与列表不同，集合没有顺序概念——既不能通过索引访问，也无法控制元素的插入位置。

### Java集合实现

Java提供了`Set`接口及`HashSet`等实现类（需导入）：

```java
import java.util.Set;
import java.util.HashSet;
```

使用示例：

```java
Set<String> s = new HashSet<>();
s.add("Tokyo");
s.add("Lagos");
System.out.println(s.contains("Tokyo")); // 真
```

### Python中的等价实现

Python直接使用`set()`：

```python
s = set()
s.add("Tokyo")
s.add("Lagos")
print("Tokyo" in s)  # 真
```
