#### 数据结构对比

|类型|特性|Java 接口|常见实现类|
|---|---|---|---|
|**List**|有序，允许重复|`List<E>`|`ArrayList`|
|**Set**|无序，元素唯一|`Set<E>`|`HashSet`|

---

#### 内置 List 使用

```java
// 导入声明
import java.util.List;
import java.util.ArrayList;

// 实例化与操作
List<Integer> numList = new ArrayList<>();
numList.add(5);       // 自动装箱 int → Integer
numList.add(10); 
System.out.println(numList); // 输出: [5, 10]
```

---

#### 内置 Set 使用

```java
// 导入声明
import java.util.Set;
import java.util.HashSet;

// 实例化与操作
Set<String> cities = new HashSet<>();
cities.add("Tokyo");
cities.add("Lagos");
System.out.println(cities.contains("Berlin")); // 输出: false
```

---

#### Python Set 对比

```python
# 创建与操作
s = set()
s.add("Tokyo")
s.add("Lagos")
print("Tokyo" in s)  # 输出: True
```

#### ArraySet 自定义实现

**核心方法**

```java
public class ArraySet<T> {
    private T[] items;
    private int size;

    // 初始化
    public ArraySet() {
        items = (T[]) new Object[100];
        size = 0;
    }

    /** 添加唯一元素 */
    public void add(T x) {
        if (contains(x)) return;
        items[size++] = x;
    }

    /** 检查元素存在性 */
    public boolean contains(T x) {
        for (int i=0; i<size; i++) {
            if (items[i].equals(x)) return true;
        }
        return false;
    }

    /** 获取元素数量 */
    public int size() {
        return size;
    }
}
```

**实现要点**

1. 泛型设计支持任意对象类型
2. 动态数组扩容需另实现（当前固定容量100）
3. `equals()` 方法依赖对象正确实现

---

#### 自定义 vs 内置集合

|特性|ArraySet|java.util.HashSet|
|---|---|---|
|底层结构|数组|哈希表|
|时间复杂度|查 O(N) 增 O(1)|查 O(1) 增 O(1)|
|空值处理|需额外逻辑|支持 null|
|内存效率|连续存储|链表/红黑树|
