#### 核心流程

Java程序执行分为两步：

1. **编译**：`.java` → `.class`（字节码）
2. **解释执行**：JVM运行字节码

![](app://7d22f0a30c8c8cfadd9bfe34557a08deef36/F:/obsidian_Markdown/My_knowledge_base/PANG/Pasted%20image%2020250516202213.png?1747398133940)

#### 终端操作示例

```bash
# 编译
$ javac HelloWorld.java  

# 运行（无需.class扩展名）
$ java HelloWorld
Hello World!
```

#### .class文件特性

- 预类型检查，增强安全性
- 执行效率更高
- 保护源代码知识产权
