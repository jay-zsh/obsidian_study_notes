#### 核心流程

Java程序执行分为两步：

1. **编译**：`.java` → `.class`（字节码）
2. **解释执行**：JVM运行字节码

![[Pasted image 20250518183058.png]]

#### 终端操作示例

```bash
# 编译 （注意代码不要含有package，ide会自动添加）
$ javac HelloWorld.java  

# 运行（无需.class扩展名）
$ java HelloWorld
Hello World!
```

#### .class文件特性

- 预类型检查，增强安全性
- 执行效率更高
- 保护源代码知识产权
