#### 代码结构解析

```java
// 类声明：文件名必须为 HelloWorld.java
public class HelloWorld {     
    // JVM 程序入口点
    public static void main(String[] args) {  
        System.out.println("Hello world!");  // 标准输出语句
    }
}
```

---

#### 核心概念说明

|组件|作用|
|---|---|
|**public class**|类访问修饰符，类名必须与文件名一致|
|**main 方法**|Java 程序唯一入口，格式固定：`public static void main(String[] args)`|
|**System.out**|标准输出流对象|
|**println()**|输出内容并换行（`print()` 不换行）|

---

#### 编译运行流程

1. **编译**（生成字节码）
    
    ```bash
    javac HelloWorld.java  # 生成 HelloWorld.class
    ```
    
2. **执行**（JVM加载运行）
    
    ```bash
    java HelloWorld        # 输出 Hello world!
    ```
    

---

#### 关键特性说明

- **大小写敏感**：`system` ≠ `System`
- **分号结尾**：每条语句必须以 `;` 结束
- **代码块界定**：用 `{}` 包裹代码块
- **注释规范**：
    
    ```java
    // 单行注释
    /* 多行注释 */
    /** Javadoc文档注释 */
    ```
    

---

#### 扩展知识

▸ 参数 `String[] args` 用于接收命令行参数  
▸ `public` 表示全局可访问性  
▸ `static` 使方法无需实例化即可调用  
▸ `void` 表示方法无返回值
