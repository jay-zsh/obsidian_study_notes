## 空间有界压缩

但空间有界压缩问题如何？该问题接受两个输入：比特流 BB 和目标大小 SS。目标是找到长度 ≤S≤S 且能输出 BB 的程序。

事实证明，该问题同样不可计算。若可解，我们只需对不同的 SS 值进行二分搜索即可找到最优压缩程序大小——而这与前一章节的结论相悖。

---

## 时空有界压缩

若在上述问题中增加"最多执行 TT 行字节码"的约束呢？

这看似违反直觉，但该问题实际可解。算法如下：

```python
for length L = 1....S:
    for each possible program P of length L:
        while (P is running && !(B is outputted) && lines_executed < T):
            run the next line of P
```

算法时间复杂度为 O(T∗2S)O(T∗2S)，最终可能输出满足约束的可行程序 `P`，或返回无解。
