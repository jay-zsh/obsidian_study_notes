# 矩阵的秩

---
### 3.2矩阵的秩 核心定义
1. **秩的定义**  
   - 矩阵$A$的秩（记作$r(A)$）是其行/列向量组的极大线性无关组中向量的个数
   - 等价定义：
     - 非零子式的最高阶数
     - 行阶梯形矩阵的非零行数

1. **基本性质** 

| 性质    | 数学表达式                              |
| ----- | ---------------------------------- |
| 秩的范围  | $0 \leq r(A_{m×n}) \leq \min(m,n)$ |
| 转置不变性 | $r(A^T) = r(A)$                    |
| 乘积的秩  | $r(AB) \leq \min(r(A),r(B))$       |
| 和的秩   | $r(A+B) \leq r(A)+r(B)$            |



2. **重要定理**  
   - **秩-零化度定理**：对于$m×n$矩阵，$r(A) + \text{nullity}(A) = n$
   - **可逆等价条件**：方阵$A$可逆 $\Leftrightarrow r(A)=n$

---

### 秩的计算方法
1. **初等变换法**  
   - 通过行/列初等变换化为阶梯形矩阵
   - 示例：
     ```
     [1 2 3]    [1  2  3]
     [2 4 6] → [0  0  0]  ⇒ r(A)=1
     [3 6 9]    [0  0  0]
     ```

2. **分块矩阵法**  
   - 利用分块矩阵求秩技巧（如$r\begin{pmatrix} A & B \\ 0 & C \end{pmatrix} \geq r(A)+r(C)$）

---

### 秩的应用场景
1. **方程组解判定**  
   - $Ax=b$有解 $\Leftrightarrow r(A)=r([A|b])$
   - 解空间的维数 = $n - r(A)$

2. **矩阵分解**  
   - 秩分解：$A_{m×n}=BC$，其中$B_{m×r}$, $C_{r×n}$，$r=r(A)$
