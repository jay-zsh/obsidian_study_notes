# 矩阵的秩
[[通俗理解秩到底是什么]]

---
### 3.2矩阵的秩 核心定义
1. **秩的定义**  
   - 矩阵$A$的秩（记作$r(A)$）是其行/列向量组的极大线性无关组中向量的个数
   - 等价定义：
     - 非零子式的最高阶数
     - ==行阶梯形矩阵的非零行数==

---

### 基本性质

1. ​**性质①**​：对于$m×n$矩阵$A$，其秩满足  
   $$0\leq R(A_{m×n})\leq\min\{m,n\}$$

2. ​**性质②**​：矩阵转置不改变秩  
   $$R(A^T)=R(A)$$

3. ​**性质③**​：矩阵等价性保持秩  
   若$A\sim B$，则$R(A)=R(B)$

4. ​**性质④**​：可逆变换保持秩  
   若$P,Q$可逆，则$R(PAQ)=R(A)$

### 秩的不等式
5. ​**性质⑤**​：分块矩阵秩的范围  
   $$\max\{R(A),R(B)\}\leq R(A,B)\leq R(A)+R(B)$$

6. ​**性质⑥**​：矩阵和的秩  
   $$R(A+B)\leq R(A)+R(B)$$

7. ​**性质⑦**​：矩阵乘积的秩  
   $$R(AB)\leq\min\{R(A),R(B)\}$$  


8. ​**性质⑧**​：零积矩阵的秩关系  
   若$A_{m×n}B_{n×l}=0$，则  
   $$R(A)+R(B)\leq n$$

[[8个性质的通俗理解]]
### **重要定理**  
   - **可逆等价条件**：方阵$A$可逆 $\Leftrightarrow r(A)=n\Leftrightarrow$A满秩

---

### 秩的计算方法
1. **初等变换法**  
   - 通过行/列初等变换化为阶梯形矩阵
   - 示例：
     ```
     [1 2 3]    [1  2  3]
     [2 4 6] →  [0  0  0]  ⇒ r(A)=1
     [3 6 9]    [0  0  0]
     ```

2. **分块矩阵法**  
   - 利用分块矩阵求秩技巧（如$r\begin{pmatrix} A & B \\ 0 & C \end{pmatrix} \geq r(A)+r(C)$）

---

### 秩的应用场景
1. **方程组解判定**  
   - $Ax=b$有解 $\Leftrightarrow r(A)=r([A|b])$

2. **矩阵分解**  
   - 秩分解：$A_{m×n}=BC$，其中$B_{m×r}$, $C_{r×n}$，$r=r(A)$
