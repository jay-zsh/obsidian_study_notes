### **基本积分公式**
1. $\int k \, dx = kx + C$  
2. $\int x^n \, dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$  
3. $\int \frac{1}{x} \, dx = \ln|x| + C$  
4. $\int e^x \, dx = e^x + C$  
5. $\int a^x \, dx = \frac{a^x}{\ln a} + C \quad (a>0, a \neq 1)$  

---

### **三角函数积分**
1. $\int \sin x \, dx = -\cos x + C$  
2. $\int \cos x \, dx = \sin x + C$  
3. $\int \tan x \, dx = -\ln|\cos x| + C$  
4. $\int \cot x \, dx = \ln|\sin x| + C$  
5. $\int \sec^2 x \, dx = \tan x + C$  
6. $\int \csc^2 x \, dx = -\cot x + C$  
7. $\int \sec x \tan x \, dx = \sec x + C$  
8. $\int \csc x \cot x \, dx = -\csc x + C$  

---

### **反三角函数积分**
1. $\int \frac{1}{\sqrt{1-x^2}} \, dx = \arcsin x + C$  
2. $\int \frac{-1}{\sqrt{1-x^2}} \, dx = \arccos x + C$  
3. $\int \frac{1}{1+x^2} \, dx = \arctan x + C$  

---

### **双曲函数积分**
1. $\int \sinh x \, dx = \cosh x + C$  
2. $\int \cosh x \, dx = \sinh x + C$  

---

### **分式积分**
1. $\int \frac{1}{x^2 + a^2} \, dx = \frac{1}{a} \arctan\left(\frac{x}{a}\right) + C$  
2. $\int \frac{1}{x^2 - a^2} \, dx = \frac{1}{2a} \ln\left|\frac{x-a}{x+a}\right| + C$  
3. $\int \frac{1}{\sqrt{a^2 - x^2}} \, dx = \arcsin\left(\frac{x}{a}\right) + C \quad (a>0)$  

---

### **换元积分法常用形式**
1. $\int f(ax+b) \, dx = \frac{1}{a} F(ax+b) + C$  
2. $\int u \, dv = uv - \int v \, du$ （分部积分法）  

---

### **其他常见积分**
1. $\int \ln x \, dx = x \ln x - x + C$  
2. $\int \frac{f'(x)}{f(x)} \, dx = \ln|f(x)| + C$
