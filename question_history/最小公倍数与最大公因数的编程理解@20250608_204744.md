---

### **1. 最大公因数（GCD）**
**定义**：两个数的最大公因数是能同时整除它们的最大正整数。  
**编程实现**：
- **欧几里得算法（递归/迭代）**：高效计算GCD的经典方法。
  ```python
  def gcd(a, b):
      while b:
          a, b = b, a % b
      return a
  ```
  **示例**：`gcd(12, 18)` 返回 `6`。

- **数学库**：Python的`math.gcd()`（注意：返回的总是非负数）。

---
### **2. 最小公倍数（LCM）**
**定义**：两个数的最小公倍数是能被它们同时整除的最小正整数。
