**user**: 什么是谓词函数
[Timestamp: 2025/06/08 20:42:38]

**ai**: 在编程和数学逻辑中，**谓词函数（Predicate Function）** 是指一个返回布尔值（`true` 或 `false`）的函数，通常用于检查某个条件是否满足。它的核心特点是：

1. **输入**：接受一个或多个参数（如变量、对象等）。
2. **输出**：返回 `true`（条件成立）或 `false`（条件不成立）。
3. **用途**：常用于条件判断、过滤数据、验证规则等场景。

### 示例（不同语言）
- **JavaScript**：
  ```javascript
  const isEven = (num) => num % 2 === 0; // 检查是否为偶数
  console.log(isEven(4)); // true
  ```

- **Python**：
  ```python
  def is_positive(n):
      return n > 0
  print(is_positive(-5)) # False
  ```

- **数学逻辑**：
  类似 $P(x)$ 表示“x 满足性质 P”，如 $Prime(x)$ 判断 x 是否为质数。

### 常见应用场景
1. **数组过滤**（如 `filter(isEven, array)`）。
2. **条件分支**（如 `if (isValid(input)) { ... }`）。
3. **断言检查**（如单元测试中的 `assert`）。
