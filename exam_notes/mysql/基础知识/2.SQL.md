# 2.1SQl通用语法

- sql语句以分号结尾，可单行或多行
- sql语句可以使用空格增强可读性g
- 不区分大小写，但建议用大写
- 注释
	- 单行 --或#
	- 多行/* 注释内容*/
## sql分类
- 定义 DDL : 定义数据库对象
- 操作 DML ：数据的增删改查
- 查询 DQL ：查询数据库中表的记录
- 控制 DCL ： 创建数据库用户，控制访问权限

## DDL-数据库操作

- 查询所有数据库
>SHOW DATABASES；

- 查询当前正在使用的数据库
>SELECT DATABASE();

- 创建
>CREATE DATABASE [IF NOT EXISTS] 数据库名 '[DEFAULT CHARSET 字符集]''[COLLATE 排序规则]'；

- 使用
USE 数据库名；

## DDL-表操作
### DDL表查询操作

- 查询当前数据库的所有表 
>SHOW TABLES;

- 查询表结构
>DESC 表名；

- 查询指定的建表语句
>SHOW CREATE TABLE 表名；

### DDL表创建操作

- 创建
>CREATE TABLE 表名(
>	字段1 字段1类型[COMMENT 字段1注释]
>	字段2 字段2类型[COMMENT 字段2注释]
>	...
>	
>)[COMMENT 表注释]；

- 数据类型(tiny small mediu long)
	- 数值类型
		- INT
		- DOUBLE
	- 字符串
		- CHAR VARCHAR 定长就用CHAR 性能好
		- BLOB(存储二进制数据的)
		- TXT(长文本)
	- 日期类型
		- DATE(birthday 2025-10-01)
		- TIME
		- YEAR
		- TIMESTAMP



### DDL表修改操作

- 添加字段
>ALTER TABLE 表名 ADD 字段名 类型（长度）[COMMENT 注释][约束]；
- 修改数据类型
>ALTER TABLE 表名 MODIFY 字段名 新数据类型（长度）；
- 修改字段名和字段类型
>ALTER TABLE 表名 CHANGE 旧字段名 新字段名 类型（长度）[COMMENT 注释][约束]；
- 删除字段
>ALTER TABLE 表名 DROP 字段名；
- 修改表名
>ALTER TABLE 表名 RENAME TO 新表名；

### DDL表删除

- 删除表
>DROP TABLE [IF EXISTS] 表名；

- 删除指定表，并重新创建该表(只有表结构没有数据)
>TRUNCATE TABLE 表名；


## DML-数据操作操作（增删改）


### DML插入（INSERT）

- DML添加数据
```sql
#1.给指定字段添加数据
	INSERT INTO 表明(字段名1,...)VALUES(值1,...);
#2.给全部字段添加数据
	INSERT INTO 表名 VALUES(值1,...);
#3.批量添加数据
	INSERT INTO 表名(字段1,...)VALUES(值1,...),(值1,...),(值1,...);
	INSERT INTO 表名 VALUES(值1,...),(值1,...),(值1,...);
	
	
```

### DML修改和删除（UPDATE,DELETE)
- DML修改数据

```sql
#1.更改数据，没有where将更改整张表格
	UPDATE 表名 SET 字段名1 = 值1,字段名2 = 值2,...[WHERE 条件];
	
#2.删除数据
	DELETE FROM 表名 [WHERE 条件]
```


## DQL-数据查询

#### DQL语法

```sql
SELECT
	字段列表
FROM
	表名列表
WHERE
	条件列表
GROUP BY
	分组字段列表
HAVING
	分组后条件列表
ORDER BY
	排序字段列表
LIMIT
	分页参数
```

#### 基本查询

- 查询多个字段
>SELECT 字段1,字段2,字段3,...FROM 表名；
>SELECT * FROM 表名；

- 设置别名

>SELECT 字段1[AS 别名], 字段2[AS 别名2]... FROM 表名；

- 去除重复记录

>SELECT DISTINCT 字段名称 FROM 表名；


#### 条件查询

- 1.语法
>SELECT 字段列表 FROM 表名 WHERE 条件列表；
