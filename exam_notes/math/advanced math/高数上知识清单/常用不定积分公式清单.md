### **基本积分公式**
1. $\int k \, dx = kx + C$  
2. $\int x^n \, dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$  
3. $\int \frac{1}{x} \, dx = \ln|x| + C$  
4. $\int e^x \, dx = e^x + C$  
5. $\int a^x \, dx = \frac{a^x}{\ln a} + C \quad (a>0, a \neq 1)$  

---

### **三角函数积分**
1. $\int \sin x \, dx = -\cos x + C$  
2. $\int \cos x \, dx = \sin x + C$  
3. $\int \tan x \, dx = -\ln|\cos x| + C$  
4. $\int \cot x \, dx = \ln|\sin x| + C$  
5. $\int \sec^2 x \, dx = \tan x + C$  
6. $\int \csc^2 x \, dx = -\cot x + C$  
7. $\int \sec x \tan x \, dx = \sec x + C$  
8. $\int \csc x \cot x \, dx = -\csc x + C$  

---

### **反三角函数积分**
1. $\int \frac{1}{\sqrt{1-x^2}} \, dx = \arcsin x + C$  
2. $\int \frac{-1}{\sqrt{1-x^2}} \, dx = \arccos x + C$  
3. $\int \frac{1}{1+x^2} \, dx = \arctan x + C$  

---

### **双曲函数积分**
1. $\int \sinh x \, dx = \cosh x + C$  
2. $\int \cosh x \, dx = \sinh x + C$  

---

### **分式积分**
1. $\int \frac{1}{x^2 + a^2} \, dx = \frac{1}{a} \arctan\left(\frac{x}{a}\right) + C$  
2. $\int \frac{1}{x^2 - a^2} \, dx = \frac{1}{2a} \ln\left|\frac{x-a}{x+a}\right| + C$  
3. $\int \frac{1}{\sqrt{a^2 - x^2}} \, dx = \arcsin\left(\frac{x}{a}\right) + C \quad (a>0)$  

---

### **换元积分法常用形式**
1. $\int f(ax+b) \, dx = \frac{1}{a} F(ax+b) + C$  
2. $\int u \, dv = uv - \int v \, du$ （分部积分法）  

---

### **其他常见积分**
1. $\int \ln x \, dx = x \ln x - x + C$  
2. $\int \frac{f'(x)}{f(x)} \, dx = \ln|f(x)| + C$  

# 求不定积分常用方法
---


---

### **一、第一类换元法（凑微分法）**
**核心思想**：通过变量替换 $u = g(x)$，将积分 $\int f(g(x))g'(x)dx$ 转化为 $\int f(u)du$。  
**关键步骤**：  
1. 识别被积函数中的复合结构 $f(g(x))$ 和其导数 $g'(x)$（或相差常数倍）。  
2. 设 $u = g(x)$，将 $g'(x)dx$ 替换为 $du$。  
3. 积分后回代 $u = g(x)$。

---

#### **例1：简单多项式复合**
计算 $\int (2x+1)^5 dx$。  
**步骤**：  
1. 设 $u = 2x + 1$，则 $du = 2dx$ → $dx = \frac{1}{2}du$。  
2. 原积分变为 $\int u^5 \cdot \frac{1}{2}du = \frac{1}{2} \cdot \frac{u^6}{6} + C$。  
3. 回代 $u$：$\frac{(2x+1)^6}{12} + C$。

---

#### **例2：三角函数复合**
计算 $\int \sin(3x) \cos(3x) dx$。  
**步骤**：  
1. 设 $u = \sin(3x)$，则 $du = 3\cos(3x)dx$ → $\cos(3x)dx = \frac{1}{3}du$。  
2. 原积分变为 $\int u \cdot \frac{1}{3}du = \frac{u^2}{6} + C$。  
3. 回代 $u$：$\frac{\sin^2(3x)}{6} + C$。

---

#### **例3：指数函数复合**
计算 $\int \frac{e^{\sqrt{x}}}{\sqrt{x}} dx$。  
**步骤**：  
1. 设 $u = \sqrt{x} = x^{1/2}$，则 $du = \frac{1}{2}x^{-1/2}dx = \frac{1}{2\sqrt{x}}dx$ → $\frac{dx}{\sqrt{x}} = 2du$。  
2. 原积分变为 $\int e^u \cdot 2du = 2e^u + C$。  
3. 回代 $u$：$2e^{\sqrt{x}} + C$。

---

### **二、第二类换元法**
**核心思想**：通过代换 $x = g(t)$ 简化被积函数（尤其是含根式、三角函数的积分）。  
**常见类型**：  
1. **三角换元**：处理 $\sqrt{a^2-x^2}$、$\sqrt{a^2+x^2}$、$\sqrt{x^2-a^2}$。  
2. **根式换元**：直接设 $t = \sqrt{ax+b}$。  
3. **倒代换**：设 $x = 1/t$ 处理分母高次项。

---

#### **例4：三角换元（$\sqrt{a^2-x^2}$ 型）**
计算 $\int \frac{dx}{\sqrt{9-x^2}}$。  
**步骤**：  
1. 设 $x = 3\sin t$（因为 $9-x^2 = 9\cos^2 t$），则 $dx = 3\cos t \, dt$。  
2. 原积分变为 $\int \frac{3\cos t}{3\cos t} dt = \int 1 \, dt = t + C$。  
3. 回代 $t$：由 $x = 3\sin t$ 得 $t = \arcsin(x/3)$，结果为 $\arcsin\left(\frac{x}{3}\right) + C$。

---

#### **例5：三角换元（$\sqrt{x^2+a^2}$ 型）**
计算 $\int \frac{dx}{\sqrt{x^2+4}}$。  
**步骤**：  
1. 设 $x = 2\tan t$（因为 $x^2+4 = 4\sec^2 t$），则 $dx = 2\sec^2 t \, dt$。  
2. 原积分变为 $\int \frac{2\sec^2 t}{2\sec t} dt = \int \sec t \, dt = \ln|\sec t + \tan t| + C$。  
3. 回代 $t$：由 $x = 2\tan t$，画直角三角形得 $\sec t = \frac{\sqrt{x^2+4}}{2}$，结果为 $\ln\left|\frac{\sqrt{x^2+4}}{2} + \frac{x}{2}\right| + C$（可合并常数）。

---

#### **例6：根式换元**
计算 $\int \frac{x}{\sqrt{2x+1}} dx$。  
**步骤**：  
1. 设 $t = \sqrt{2x+1}$，则 $x = \frac{t^2-1}{2}$，$dx = t \, dt$。  
2. 原积分变为 $\int \frac{(t^2-1)/2}{t} \cdot t \, dt = \frac{1}{2} \int (t^2-1) dt = \frac{t^3}{6} - \frac{t}{2} + C$。  
3. 回代 $t$：$\frac{(2x+1)^{3/2}}{6} - \frac{\sqrt{2x+1}}{2} + C$。

---

#### **例7：倒代换**
计算 $\int \frac{dx}{x\sqrt{x^2-1}}$（$x > 1$）。  
**步骤**：  
1. 设 $x = \frac{1}{t}$，则 $dx = -\frac{1}{t^2}dt$，$\sqrt{x^2-1} = \frac{\sqrt{1-t^2}}{|t|}$。  
2. 原积分变为 $\int \frac{-1/t^2}{(1/t)\cdot(\sqrt{1-t^2}/t)} dt = -\int \frac{dt}{\sqrt{1-t^2}} = -\arcsin t + C$。  
3. 回代 $t = 1/x$：$-\arcsin\left(\frac{1}{x}\right) + C$（或写作 $\arccsc x + C$）。

---

### **总结对比**
| **方法**       | **适用场景**                     | **关键步骤**                     |
|----------------|----------------------------------|----------------------------------|
| **第一类换元** | 被积函数含复合函数 $f(g(x))$     | 凑出 $du = g'(x)dx$              |
| **第二类换元** | 含根式、三角项或分母高次         | 选择代换 $x=g(t)$ 消去根式/简化  |

**练习建议**：  
1. 第一类换元法多练习“凑微分”的观察能力。  
2. 第二类换元法需熟悉三角恒等式和回代技巧。
