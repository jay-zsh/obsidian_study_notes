### 1. 基本概念
- **向量**：有大小和方向的量，记作 $\vec{a}$  
- **单位向量**：模为1的向量，$\vec{a}$ 的单位向量 $\vec{a}^0 = \frac{\vec{a}}{|\vec{a}|}$  
- **夹角**：向量 $\vec{a}$ 与 $\vec{b}$ 的夹角 $\theta$ 满足 $\cos \theta = \frac{\vec{a} \cdot \vec{b}}{|\vec{a}| |\vec{b}|}$  
- **共面**：若多个向量可置于同一平面，则它们共面（标量三重积 $\vec{a} \cdot (\vec{b} \times \vec{c}) = 0$）

### 2. 向量的线性运算
- **三角形法则**：$\vec{a} + \vec{b}$ 为以 $\vec{a}$、$\vec{b}$ 为邻边的对角线  
- **交换律**：$\vec{a} + \vec{b} = \vec{b} + \vec{a}$  
- **结合律**：$(\vec{a} + \vec{b}) + \vec{c} = \vec{a} + (\vec{b} + \vec{c})$  
- **负向量**：$-\vec{a}$ 与 $\vec{a}$ 大小相等、方向相反  
- **数乘**：$k \vec{a}$（$k>0$ 同向，$k<0$ 反向）  
- **定理1**：$\vec{a} \parallel \vec{b} \iff \exists \lambda \ne 0, \vec{a} = \lambda \vec{b}$  

### 3. 空间直角坐标系
- 坐标系：$Oxyz$ 三轴互相垂直  
- 点坐标：$M(x,y,z)$  
- 基向量：$\vec{i}=(1,0,0),\ \vec{j}=(0,1,0),\ \vec{k}=(0,0,1)$  

### 4. 坐标下的线性运算
设 $\vec{a} = (a_x, a_y, a_z),\ \vec{b} = (b_x, b_y, b_z)$  
- **加法**：$\vec{a} + \vec{b} = (a_x + b_x, a_y + b_y, a_z + b_z)$  
- **数乘**：$k \vec{a} = (k a_x, k a_y, k a_z)$  
- **平行条件**：$\frac{a_x}{b_x} = \frac{a_y}{b_y} = \frac{a_z}{b_z}$（若 $b_i \ne 0$）

### 5. 向量的模、方向角与投影
#### (1) 向量的模与距离公式
- 模：$|\vec{a}| = \sqrt{a_x^2 + a_y^2 + a_z^2}$  
- 两点距离：$|M_1 M_2| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2}$  

#### (2) 方向角与方向余弦
- 方向角：$\alpha, \beta, \gamma$（与 $x,y,z$ 轴夹角）  
- 方向余弦：  
  $\cos \alpha = \frac{a_x}{|\vec{a}|},\ \cos \beta = \frac{a_y}{|\vec{a}|},\ \cos \gamma = \frac{a_z}{|\vec{a}|}$  
  满足 $\cos^2 \alpha + \cos^2 \beta + \cos^2 \gamma = 1$  

#### (3) 向量在轴上的投影
- **投影**：$\text{Prj}_u \vec{a} = |\vec{a}| \cos \theta$（$\theta$ 为 $\vec{a}$ 与 $u$ 轴夹角）  
- **性质**：  
  1. $\text{Prj}_u (\vec{a} + \vec{b}) = \text{Prj}_u \vec{a} + \text{Prj}_u \vec{b}$  
  2. $\text{Prj}_u (k \vec{a}) = k \cdot \text{Prj}_u \vec{a}$  
  3. $\text{Prj}_u \vec{a} = \vec{a} \cdot \vec{u}^0$（$\vec{u}^0$ 为 $u$ 轴单位向量）
